# Guantlet like
A simple gauntlet-like clone made in Godot. Explore the maze, find keys, kill the mobs and move to the next floor.

## Scope
- Single-player
- 10 Levels
- 5 Enemy Types
- 1 spell
- Small Keys unlock doors
- Floor Key unlocks exit
- Arcade style points
- Local high score/Top 5 scores

## Defined Scope Creep
- Local Coop
- Proc Gen Levels
- Spell Upgrades
- Score Multipliers
- Local Leaderboard

## Undefined Scope Creep
- Multiple more enemies
- More complex AI
- Multiple spell types
- Network Multiplayer
- AOE speels
- Bosses
